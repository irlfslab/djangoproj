About this project

Introduction


This project is to design a website to display records from Django models using test data.

The Home page displays a list of staff records. When the name is selected, it shows the details for any records selected from the home page.

Features


On the homepage, it shows the summary of staff appointment. The data is preset in database.

When the url of name is selected, it will direct to other page which will display the staff info.


Technologies used in web development


  • HTML5
  • CSS3
  • Bootstrap
  • Jquery
  • Javascript
  • Python
  • Django
